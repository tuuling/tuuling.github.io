Sending jQuery as AJAX response from PHP backend

This approach enables you to seamlessly embed your back-end PHP generated responses to your front-end Javascript. You can write jQuery in PHP using PHP syntax and send it through a small and lightweight middle layer which then executes your script without the need to write a parser or translator for your responses. It is very easy to implement and fast to learn, you only need to know some basic PHP and jQuery syntax. I have found that this method greatly reduces my development time and gives me a lot more freedom implementing different and more complex front-end interfaces.

The plugin used for this is made by Anton Shevchuk. You can view his webpage at http://anton.shevchuk.name and the page for the plugin at http://jquery.hohli.com/

For this example i'm using minimal HTML5 syntax. You will need only one HTML file for this to work. So use this markup and save it as index.html at your document root folder. For this this example PHP is only used to generate Ajax responses and is not needed for inital page load.


<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="utf-8" />
	</head>
	<body>
	
	</body>
</html>

#2 Add jQuery to you page

First thing we need is to add jQuery framework support to our page. I prefer to add it from Google Code. Or you can download it from http://docs.jquery.com/Downloading_jQuery and host it from your own server.


<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="utf-8" />
		<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>
	</head>
	<body>
	
	</body>
</html>
